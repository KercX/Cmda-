# Cmda

![Cmda](https://github.com/KercX/Cmda/blob/main/assets/IMG_9406.jpeg)


**Cmda** is a modern, lightweight, and highly extensible command framework for Roblox, written in Luau.

Its not copy Cmdr please do not ban its my project okay, Cmda was built from the ground up with a focus on simplicity, performance, and ease of extension.

## Features

- Fully typed argument system with autocomplete support
- Client and server-side command validation
- Powerful hooks system (BeforeRun, AfterRun, and more)
- Easy creation of custom argument types
- Clean and modern console UI
- Designed with Rojo and Wally in mind
- Clear and maintainable architecture

## How It Works

Cmda allows developers to register custom commands, argument types, and hooks.  
The console can be opened in-game using the **\`** (backquote) key.

Everything is built to be modular, making it simple to extend and customize for any Roblox experience.

## Planned Improvements

- Full argument parser with quote support
- Tab autocomplete
- Command history (arrow keys)
- Additional built-in types
- TextChatService integration
